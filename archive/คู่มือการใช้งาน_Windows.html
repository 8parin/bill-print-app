ขั้นตอนที่ 1 — เปิด Command Prompt

1

กด Win + R แล้วพิมพ์ cmd แล้วกด Enter

(หรือกด Start แล้วค้นหา "Command Prompt")

2

ใช้คำสั่ง cd เพื่อเข้าโฟลเดอร์ที่แตกไฟล์ไว้ เช่น:

cd C:\Users\ชื่อของคุณ\Documents\Batch_Bill_Print_v1.5_windows
💡 เปลี่ยน ชื่อของคุณ เป็นชื่อ User บนเครื่อง และเปลี่ยน path ให้ตรงกับที่แตกไฟล์

ขั้นตอนที่ 2 — ตรวจสอบ Python

คัดลอกคำสั่งนี้วางใน Command Prompt แล้ว Enter:

python --version
✅ ถ้าเห็น Python 3.x.x ข้ามไปขั้นตอนที่ 3 ได้เลย
❌ ถ้าขึ้น "'python' is not recognized" แสดงว่ายังไม่ได้ติดตั้ง Python

ให้ดาวน์โหลด Python จาก https://www.python.org/downloads/
ระหว่างติดตั้ง ต้องเช็กกล่อง "Add Python to PATH" ด้วย แล้วกลับมารันคำสั่งข้างบนอีกครั้ง

ขั้นตอนที่ 3 — ติดตั้งโปรแกรม (ครั้งแรกครั้งเดียว)

คัดลอกคำสั่งนี้วางใน Command Prompt แล้ว Enter:

install.bat

รอประมาณ 2-5 นาที จนเห็น "[OK] Installation Complete!"

⚠️ ถ้าติดตั้งไม่ผ่าน ให้ลองรันคำสั่งนี้แทน:
python -m pip install -r requirements.txt

ขั้นตอนที่ 4 — เปิดโปรแกรม

วิธีที่ 1 (ง่ายที่สุด) — ดับเบิ้ลคลิกไฟล์ BatchBill.bat บน Desktop

วิธีที่ 2 — รันใน Command Prompt:

run.bat

แล้วเปิดเบราว์เซอร์ไปที่:

http://localhost:5003
💡 ปิดโปรแกรม: กด Ctrl + C ใน Command Prompt

🛠️ แก้ปัญหาเบื้องต้น

ปัญหาวิธีแก้
"python is not recognized"ติดตั้ง Python ใหม่ และเช็กกล่อง "Add Python to PATH"
Port 5003 ถูกใช้อยู่ปิดหน้าต่าง CMD ที่รันอยู่ แล้วเปิดใหม่
ติดตั้งไม่ผ่าน (Error)เปิด CMD แบบ "Run as Administrator" แล้วรัน install.bat ใหม่
เว็บไม่เปิดเปิด Chrome/Edge แล้วพิมพ์ http://localhost:5003
