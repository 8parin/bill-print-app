ขั้นตอนที่ 1 — เปิด Terminal

1

กด Cmd + Space แล้วพิมพ์ Terminal แล้วกด Enter

2

ลากโฟลเดอร์ Batch_Bill_Print_v1.5_mac จาก Finder มาวางใน Terminal แล้วกด Enter
(หรือพิมพ์ cd แล้วลากโฟลเดอร์ใส่)

ขั้นตอนที่ 2 — ตรวจสอบ Python

คัดลอกคำสั่งนี้วางใน Terminal แล้ว Enter:

python3 --version
✅ ถ้าเห็น Python 3.x.x ข้ามไปขั้นตอนที่ 3 ได้เลย
❌ ถ้าขึ้น "command not found" ให้ติดตั้ง Python ก่อน:
brew install python3
💡 ถ้าไม่มี Homebrew ให้ดาวน์โหลด Python จาก https://www.python.org/downloads/ แทน

ขั้นตอนที่ 3 — ติดตั้งโปรแกรม (ครั้งแรกครั้งเดียว)

คัดลอกคำสั่งนี้วางใน Terminal แล้ว Enter:

./install.sh

รอประมาณ 2-5 นาที จนเห็น "✅ Installation Complete!"

⚠️ ถ้าขึ้น "Permission denied" ให้รันคำสั่งนี้ก่อน:
chmod +x install.sh run.sh

แล้วรัน ./install.sh อีกครั้ง

ขั้นตอนที่ 4 — เปิดโปรแกรม

วิธีที่ 1 (ง่ายที่สุด) — ดับเบิ้ลคลิกไอคอน BatchBill.command บน Desktop

วิธีที่ 2 — รันใน Terminal:

./run.sh

แล้วเปิดเบราว์เซอร์ไปที่:

http://localhost:5003
💡 ปิดโปรแกรม: กด Ctrl + C ใน Terminal

🛠️ แก้ปัญหาเบื้องต้น

ปัญหาคำสั่งแก้ไข
Permission deniedchmod +x install.sh run.sh
Port 5003 ถูกใช้อยู่kill $(lsof -ti:5003)
ติดตั้ง dependencies ไม่ผ่านpip3 install -r requirements.txt
